# themeparks

The Python SDK for the ThemeParks.wiki API.

**v2 is under active development.** A full README will land with the v2.0.0 release.
For the current stable release (v1), install `themeparks` pinned to `<2` — note that v1 has
known issues with nullable queue fields (see GitHub issues #1 and #2) which v2 fixes.

- API documentation: https://api.themeparks.wiki
