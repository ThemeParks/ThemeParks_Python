# Raw client

Low-level HTTP wrappers. You usually don't need these directly — use
[`ThemeParks`](client.md) or the [entity helpers](entity.md) instead. They're
exposed for cases where you want to call an endpoint that doesn't yet have an
ergonomic wrapper.
